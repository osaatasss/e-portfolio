01About

I'm a software engineering student who got hooked on the backend — the moment I realised most of the interesting problems live behind the API, I stopped fighting CSS and started designing schemas.

I like clean architecture, message-driven systems, and databases that don't lie to you. Lately that means a lot of ASP.NET Core and Spring Boot, a healthy respect for LavinMQ, and the occasional FastAPI service when I want to move fast.

Off the keyboard I read about distributed systems, go to dev meetups, and over-engineer side projects for fun.
